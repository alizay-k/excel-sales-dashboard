# Sales Performance Dashboard — Excel

An interactive Excel dashboard analyzing retail sales performance, built to 
practice pivot tables, slicers, and dashboard design.

## What it covers
- KPI summary cards: Total Sales, Total Profit, Avg Profit Margin, Orders
- Monthly sales trend analysis
- Sales and profit breakdown by region
- Profit by category and sub-category
- Top 5 customers by sales

## Skills applied
- Pivot tables and PivotCharts
- Interactive slicers (Region, Category, Sales Tier)
- Dashboard layout and KPI design
